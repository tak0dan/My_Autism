# nixos-build — NixOS Configuration Directory

This directory contains the full modular NixOS configuration.

It is designed to be placed at `/etc/nixos/` (or symlinked/copied there).

---

## Feature Flags

`configuration.nix` uses a `features` block to toggle entire subsystems without commenting or deleting anything:

```nix
features = {
  hyprland      = true;    # Wayland compositor + Hyprland packages
  kde           = true;    # KDE runtime (not full Plasma desktop)
  steam         = true;    # Steam + GameMode + gaming packages
  virtualisation = false;  # Docker + VirtualBox (slow to build)
  nixorcist     = true;    # Package management tooling
  openssh       = true;    # Remote SSH access
  uwu           = true;    # Aesthetic/meme extras
};
```

Setting a feature to `false` completely removes it from the build.

## Module System

Modules are imported unconditionally or behind feature flags:

```nix
imports = [
  ./hardware-configuration.nix
  ./modules/bootloader.nix
  ./modules/audio.nix
  # ...
  ./nixorcist/generated/all-packages.nix
]
++ lib.optionals features.hyprland [ ./modules/window-managers.nix ]
++ lib.optionals features.virtualisation [ ./modules/virtualbox.nix ];
```

**Rule:** If you don't understand a module, disable its feature flag — do not delete it.

## Package Groups

Each file in `packages/` is independent and self-contained. Some packages deliberately appear in multiple groups so groups stay portable (no hidden cross-dependencies).

The `disabled/disabled-packages.nix` file acts as a global filter applied at build time. Manage it with:

```bash
nixos-comment <package>     # disable
nixos-uncomment <package>   # re-enable
```

After changes, rebuild with:

```bash
nixos-smart-rebuild
```

## Nixorcist Integration

`nixorcist/generated/all-packages.nix` is imported directly in `configuration.nix`. Nixorcist manages all files inside `nixorcist/generated/` — do not edit them by hand.

For Nixorcist documentation → [nixorcist/README.md](nixorcist/README.md)

## Nvidia GPU

If your machine has an Nvidia GPU, replace the kernel params module:

```nix
# In configuration.nix imports, comment out:
# ./modules/kernel-params.nix

# And uncomment:
./modules/kernel-params-nvidia.nix
```

Review the Nvidia module and adjust parameters for your specific GPU.  
When in doubt, pull kernel parameters from a minimal known-good `configuration.nix` rather than using generic ones blindly.
